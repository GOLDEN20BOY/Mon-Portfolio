Portfolio Web — Site vitrine personnel (GitHub Pages)

Site portfolio conçu pour présenter mes projets, mes compétences et mes contacts de manière claire et professionnelle. Déployé sur GitHub Pages, il sert de vitrine pour le recrutement et pour démontrer mes réalisations.

Objectifs

  • Présenter mes projets et expériences de façon structurée
  • Offrir une navigation simple (sections + pages projets)
  • Mettre en avant une identité visuelle sobre et moderne
  • Rendre le site responsive (mobile / desktop)

Fonctionnalités

  • Mode sombre (Dark Mode)
  • Filtres de projets par catégories
  • Pages projets détaillées + carrousels d’images
  • Formulaire de contact (Formspree)
  • Base prête pour la traduction (i18n)

Technologies

  • HTML5 / CSS3 (responsive)
  • JavaScript (UI, interactions, filtres, animations)
  • GitHub Pages (déploiement)
