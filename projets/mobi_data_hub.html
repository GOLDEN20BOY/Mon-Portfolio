MOBI DATA HUB — Architecture Cloud & Gouvernance des Données

Conception et déploiement d’un système centralisé de gestion des données pour MOBI Senegal Holding. L’architecture repose sur Google Sheets (cloud) renforcé par Google Apps Script afin d’assurer un accès en ligne instantané, des mises à jour en temps réel, une sécurisation multi-niveaux et une exploitation directe via Python / API / Power BI.

Contexte

La Direction Générale souhaitait une solution fiable pour mettre en ligne les données stratégiques : finances, clients/prospects, synthèses, données confidentielles et suivi opérationnel. Le système devait être rapide à déployer, sécurisé, collaboratif et évolutif vers une architecture plus avancée (PostgreSQL / ERP).

Accès en ligne & collaboration temps réel

  • Accès sécurisé depuis n’importe quel appareil connecté (PC / mobile)
  • Mises à jour visibles instantanément par les utilisateurs autorisés
  • Travail collaboratif simultané sans conflit de versions
  • Historique complet des modifications et traçabilité
  • Disponibilité continue sans installation locale

Objectifs stratégiques

  • Centraliser les données critiques de l’entreprise
  • Mettre en place une gouvernance claire par rôle
  • Réduire les erreurs de saisie
  • Permettre l’exploitation BI et les automatisations
  • Préparer la migration vers une base SQL future

Architecture & Structuration

  • Tables organisées par domaine métier
  • Séparation données brutes / calculs / synthèses
  • Clés structurées pour logique pseudo-relationnelle
  • Organisation par service selon responsabilités

Sécurité & Gouvernance des accès

  • Partage personnalisé par invitation email
  • Droits différenciés : lecture / commentaire / édition
  • Protection avancée des feuilles et plages sensibles
  • Séparation des données confidentielles

Qualité & Intégrité des données

  • Validation de données (menus fixes, formats imposés)
  • Mise en forme conditionnelle (alertes visuelles)
  • Standardisation des champs pour éviter les incohérences
  • Réduction significative des erreurs humaines

Automatisation & Intelligence

  • Scripts automatisés via Google Apps Script
  • Synchronisation et contrôles logiques automatiques
  • Formules avancées : ARRAYFORMULA, QUERY, IMPORTRANGE
  • Tableaux dynamiques et synthèses KPI automatisées

Exploitation & Scalabilité

  • Connexion directe à Power BI
  • Extraction via API Google & Python
  • Intégration possible dans plateformes web
  • Base prête pour migration PostgreSQL / ERP

Impact

  • Centralisation des données stratégiques
  • Accès temps réel pour la Direction et les équipes
  • Amélioration de la prise de décision
  • Gain de temps opérationnel
  • Fondation pour transformation digitale interne
