Analyse de données avec Python

Résumé

Utilisation de bibliothèques Python comme pandas, seaborn et matplotlib pour analyser et visualiser des données. Création de graphiques, exploration de corrélations et résumé statistique de datasets réels.

Technologies utilisées

  • Python
  • Pandas
  • Matplotlib
  • Seaborn
  • Jupyter Notebook

Ce que j'ai appris

Manipulation de données complexes, visualisation efficace, interprétation de données statistiques, et présentation claire des résultats.
